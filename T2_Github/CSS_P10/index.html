ASIR

El ciclo formativo de grado medio "Administración de Sistemas Informaticos en Red" tiene una duración de 2000 horas que se imparten a lo largo de dos cursos lectivos. Este título se estableció mediante el Real Decreto 4/2003 publicado en el BOE nº 124 de 24/05/03, y el currículo se estableció mediante el Real Decreto 9/2003 publicado en el BOE nº 192 de 12/08/03.

Se puede acceder al ciclo siempre que se supere la correspondiente prueba de acceso (se puede solicitar la exención de esta prueba si se tiene superada la prueba de acceso a la universidad para mayores de 25 años).
